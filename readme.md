#####  找伙伴泊车app后端   

######  api     
    安卓交互接口，包括用户注册登录，车位预约。停车场管理等功能
######  domain  
    用于存放需要的属性   
######  interceptor  
     拦截器，拦截api判断是否通过验证   
######  mapper   
     操作数据库层  

###### properties   
      一些属性   
###### result   
       api返回值包装   
###### service  
       service层   
###### tcpserver  
       tcp服务器，用于与硬件交互   
###### utils  
      封装的工具
